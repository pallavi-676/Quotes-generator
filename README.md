# ✨ Random Quote Generator

A beautiful and minimal **Random Quote Generator** built using **HTML, CSS, and JavaScript**.  
It displays inspirational quotes with smooth animations and a modern aesthetic interface.

</div>

---

## 🌐 Preview

This project shows a motivational quote inside a modern card UI.  
When the user clicks the **New Quote** button, a new random quote appears with a smooth fade animation.

---

## 🚀 Features

✨ Random quote generation  
✨ Smooth fade animation when changing quotes  
✨ Glassmorphism UI design  
✨ Animated gradient background  
✨ Responsive layout for mobile and desktop  
✨ Clean and beginner-friendly code  

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

Random-Quote-Generator

## ⚙️ How It Works

1. Quotes are stored inside a JavaScript array.
2. When the **New Quote** button is clicked:
3. JavaScript selects a random quote.
4. The quote text and author update dynamically with animation.
